# BUG-102: API_BASE hardcoded to localhost:3000

### Severity
- [ ] S1 — Critical
- [ ] S2 — High
- [x] S3 — Medium
- [ ] S4 — Low

### Environment
| Field | Value |
|-------|-------|
| Browser | All |
| OS | All |
| Viewport | All |

### Steps to Reproduce
1. Open `frontend/js/api.js`
2. See line 1: `const API_BASE = 'http://localhost:3000/api';`

### Expected Behavior
The API base URL should be configurable for different deployment environments (production, staging, etc.)

### Actual Behavior
Hardcoded to localhost:3000. Will fail when deployed to an actual operator's domain.

### Fix
Use a relative path or read from a config/environment variable:
```javascript
const API_BASE = window.API_BASE || '/api';
```
Or set via a `<meta>` tag or script config in the HTML.
